# 🧠 Deep Learning Projects

## 📌 Overview

This repository contains a collection of **Deep Learning projects** ranging from fundamental concepts to advanced applications, with a strong focus on **computer vision** and **medical image analysis**.

The goal is to provide practical, real-world implementations that bridge the gap between theory and application.

---

## 🚀 Projects

### 1️⃣ Chest X-Ray Classification

* **Task:** Classify chest X-ray images (e.g., Pneumonia vs Normal)
* **Model:** Convolutional Neural Networks (CNN)
* **Key Techniques:**

  * Data preprocessing & augmentation
  * Model training and evaluation
  * Grad-CAM for model interpretability

---
